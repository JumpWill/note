# 华为云 (Huawei Cloud)

## 计算

- **ECS**：弹性云服务器，规格族 通用 / 计算 / 内存 / 鲲鹏 (ARM) / 昇腾 (NPU)
- **BMS**：裸金属
- **FunctionGraph**：函数计算
- **HECS**：专属主机

## 存储

- **OBS**：对象存储，标准 / 低频 / 归档 / 深度归档
- **EVS**：云硬盘 (SSD / 极速 SSD)
- **SFS**：弹性文件服务 (NFS)
- **SFS Turbo**：高性能并行文件存储

## 数据库

- **RDS**：MySQL / PG / SQL Server
- **GaussDB**：自研 (基于 openGauss)，分集中式 / 分布式 (DWS)
- **GaussDB (for MySQL/PG)**：兼容版
- **DCS**：托管 Redis / Memcached
- **DDS**：托管 MongoDB
- **DDM**：分布式数据库中间件
- **GeminiDB**：多模 (Cassandra / Redis / Influx / Mongo)

## 网络

- **VPC**：私有网络 + 子网
- **ELB**：弹性负载均衡 (七层 / 四层)
- **EIP / NAT GW / 共享带宽**
- **CC (云连接)**：跨地域 VPC 互通
- **VPN / DC (专线)**
- **CDN**
- **VPCEP**：终端节点服务

## 容器

- **CCE**：云容器引擎 (CCE Standard / Turbo)
- **CCE Autopilot**：Serverless K8s
- **SWR**：镜像仓库
- **ASM**：应用服务网格

## 安全与身份

- **IAM**：身份认证 (联邦 / SSO / RBAC)
- **KMS / CSMS / DEW**：数据加密服务
- **WAF / Anti-DDoS / HSS (主机安全)**
- **DBSS / CodeArts Inspector**

## 监控运维

- **AOM / APM**：应用运维管理 + 应用性能
- **LTS**：日志服务
- **CES**：云监控 (告警 / 指标)
- **CFT (应用运维) / APM**

## 消息

- **DMS (分布式消息服务)**：Kafka / RabbitMQ / RocketMQ
- **EventGrid**：事件网格

## IaC

- **RFS (资源编排服务)**：Terraform 兼容
- **Terraform**
